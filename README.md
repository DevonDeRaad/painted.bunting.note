# painted.bunting.note

This repository holds the code needed to reproduce the analyses presented in the manuscript "Painted Bunting (Passerina ciris) breeding in South Dakota suggests potential northward expansion in response to ongoing anthropogenically induced climate change"

### Breeding Bird Survey abundance data
Detailed breakdown of constructing and visualizing Bayesian abundance models based on Breeding Bird Survey data can be found here: https://devonderaad.github.io/painted.bunting.note/bbs2.html

### Climate vagrancy plots
Detailed breakdown of plotting vagrancy in Nebraska and South Dakota versus annual temperature can be seen here: https://devonderaad.github.io/painted.bunting.note/climate.vagrancy.html
